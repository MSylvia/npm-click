=====================
Comparing GitGub Repositories

### Prerequisites

 - ReactJS
 - NPM
 - Grunt

### Installation
You will need `http-server` installed globally to serve the website. If you encounter any issues with `npm install`, then run `ulimit -n 512`.

    npm install -g http-server
    npm install

### Development

    npm run watch
    npm run serve
